# 💉🐶Veterinary-Appointment-Scheduling 
## 💻Feature ของระบบ
- ผู้ช่วยสามารถ สร้าง/แก้ไข/ยกเลิก(ก่อน 2 ชั่วโมง) นัดได้
- สัตวแพทย์สามารถดูตารางนัดของตนเองได้
- สัตวแพทย์สามารถอัปเดตสถานะการรักษาได้
- ผู้ช่วยดูรายการนัดหมายทั้งหมด พร้อมระบบกรอง (Filter) ตามวันหรือสัตวแพทย์
- ผู้ช่วยจำกัดจำนวนสัตว์เลี้ยงต่อช่วงเวลา ป้องกันการนัดซ้ำซ้อน
- แอดมินจัดการข้อมูลบุคลากร
- แอดมินจัดการและเพิ่มช่วงเวลาทำงาน (Slots) ของสัตวแพทย์
- ระบบเข้าสู่ระบบ (Login) และออกจากระบบ (Logout)
- ตรวจสอบเวลาว่างของสัตวแพทย์แต่ละคน
## 🛜หน้าที่ที่รับผิดชอบ
ภูมิภากร โกเมนไปรรินทร์ ทำเส้น ดูรายชื่อสัตวแพทย์ทั้งหมด และ ดูช่วงเวลาว่างของสัตวแพทย์แต่ละท่าน

ทัศน์พล พารา ทำเส้น  เข้าสู่ระบบ (บุคลากรเท่านั้น) และ ยกเลิกนัดหมาย (ต้องก่อน 2ชม.)

บุณยานุช นวลสุวรรณ์ ทำเส้น แก้ไขนัดหมาย
